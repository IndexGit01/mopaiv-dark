Install Mopaiv Dark Mode

Made by Index. Hope you enjoy!
  • Theme Download Instructions
  • - Download the Stylus Browser Extension (download links ->)
  • - Go to https://userstyles.world/style/2998/mopaiv-unofficial-dark-theme
  • - Click the "Install" button on the theme.
  • - Go to https://mopaiv.com/
  • Stylus Browser Extension Download Links
  • Links below link to the Stylus browser extension on each modern-day browser.
  • Google Chrome: https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne?hl=en
  • Microsoft Edge: https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne?hl=en
  • Opera Browser: https://addons.opera.com/en/extensions/details/stylus/
  • Firefox Browser: https://addons.mozilla.org/en-US/firefox/addon/styl-us/
  • Quick Note about Microsoft Edge: The download link for Stylus on Microsoft Edge is the same as Google Chrome, to install it on Microsoft Edge you must enable the setting "Allow extensions from other stores" or something like that (setting name may not be correct, apologies).
